# オリジナル vs 現在実装 ギャップ分析

このドキュメントでは、オリジナルの VB.NET 実装 (`original/CRUDExplorer2011`) と現在の C#/Avalonia 実装 (`src/CRUDExplorer.UI`) を詳細に比較し、未実装や不完全な機能を洗い出します。

**作成日**: 2026-04-05  
**最終更新**: 2026-04-05

---

## 凡例

| ステータス | 意味 |
|-----------|------|
| ✅ 完了 | 機能が完全に実装され、テストも完了 |
| ⚠️ 不完全 | 基本機能はあるが、一部動作に問題あり |
| ❌ 未実装 | 機能が実装されていない |
| 🔧 対応中 | 現在実装作業中 |

## 実装優先度

### 高優先度 (High)

1. **CRUD一覧のコンテキストメニュー追加**
   - コピー機能（選択行/全行）
   - Grep起動機能
   - テーブル定義表示

2. **CRUD一覧の複数列表示対応**
   - ListBoxからDataGridに変更検討
   - または複数情報をフォーマット表示

3. **FileListWindowのクエリ一覧表示**
   - ファイル選択時にクエリ一覧を右ペインに表示

4. **CrudSearchWindowにエディタ/クエリ解析ボタン追加**

5. **GrepWindowにクエリ解析ボタン追加**

6. **MainWindowにGrep/CrudSearchメニュー追加**

### 中優先度 (Medium)

7. **論理名/物理名切替ボタン**
   - 列ヘッダの表示名を切り替え

8. **FilterWindowのアクセス連動ComboBox**
   - プログラム選択でテーブルを自動絞込
   - テーブル選択でプログラムを自動絞込

9. **TableDefinitionWindowのコンテキストメニュー**
   - CrudSearchWindow起動

10. **列ヘッダクリック時のソート処理改善**

### 低優先度 (Low)

11. **列ヘッダ縦書き表示**（Avalonia制限あり）

12. **括弧マッチハイライト**

13. **デモモード表示**（認証方式変更により不要）
